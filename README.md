# Design and Analysis of Algorith using C
This repository contains a well-structured collection of Design and Analysis of Algorithms (DAA) programs written in C.  The focus is on understanding algorithmic design techniques, analyzing time and space complexity, and applying concepts to real academic problems.

This repository contains a structured and easy-to-understand collection of Design and Analysis of Algorithms (DAA) implemented in C programming.

## Who should use this repository?
- MCA / BCA / CS students
- IGNOU learners (MCS211 / MCSL216 / DAA related courses)
- Beginners learning algorithms through C
- Students preparing for practical & viva exams
- Faculty members for teaching reference

## Technologies used
 - Language: C Programming
 - Paradigm: Procedural Programming
 - Compiler: GCC / Turbo C / Any standard C compiler

## Topics covered
 - Sorting Algorithms
 - Searching Algorithms
 - Greedy Algorithms
 - Dynamic Programming
 - Graph Algorithms
 - String Matching Algorithms

## Each C program in this repository includes
- Problem statement
- Algorithm steps (written clearly, not pseudo-code heavy)
- Complete C implementation
- Sample input and output
- Time complexity analysis
- Space complexity analysis
- Important exam notes (where applicable)

## Academic use disclaimer
This repository is created for educational and learning purposes only.
Students are encouraged to understand the logic and not blindly copy the code for exams or assignments.
